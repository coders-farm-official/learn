### New Side Quest — Your Own Coding Assistant
- Added a new side quest under "Build Something New" that teaches students to run a local LLM using Ollama
- Covers how LLMs work (tokens, parameters, quantization), hardware requirements across tiers (Raspberry Pi to desktop), installing and using Ollama, the REST API, and building a web-based chat interface with HTML/CSS/JS
- Includes 3 quiz questions and a deliverable (working chat interface with custom system prompt)
- Difficulty: Intermediate | Estimated time: 5–6 hours
